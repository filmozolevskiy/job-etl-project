# UI Mockups - Job Search Campaign Management

This directory contains HTML mockups of the actual UI pages from the Campaign Management application.

## Pages Included

1. **dashboard.html** - All Campaigns list page
   - Shows table of all campaigns with status, runs, and actions
   - Includes "Run All DAGs" button
   - Displays campaign details in a table format

2. **jobs_page.html** - Jobs listing page
   - Shows jobs table with company name, status, job posting links
   - Includes status dropdowns for each job
   - Perfect fit scores and notes functionality
   - Back button to return to campaign

3. **create_campaign.html** - Create new campaign form
   - Comprehensive form with all campaign fields
   - Search criteria inputs
   - Salary range and currency selection
   - Checkbox groups for preferences
   - Ranking weights section (optional)
   - Form validation indicators

4. **view_campaign.html** - Campaign details page
   - Campaign information display
   - Action buttons (Edit, View Jobs, Run DAG, Delete)
   - Statistics grid showing campaign metrics
   - Search criteria table
   - Timestamps information

## How to View

Simply open any of these HTML files in your web browser to see the actual UI design and layout.

## Design System

All pages use the same design system:
- **Primary Color**: #007bff (Blue)
- **Success Color**: #28a745 (Green)
- **Danger Color**: #dc3545 (Red)
- **Secondary Color**: #6c757d (Gray)
- **Background**: #f5f5f5 (Light Gray)
- **Cards**: White with subtle shadows
- **Typography**: System font stack (-apple-system, Segoe UI, Roboto)

## Features Shown

- Responsive layout with max-width container
- Consistent header navigation
- Card-based content areas
- Table layouts with hover effects
- Button styles and states
- Form inputs and validation
- Status badges
- Statistics displays
